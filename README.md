# Research-Scientist-1-Code-Example
This project was in collaboration with a collegue of mine, but all of the functions uploaded are my own work. I've excluded the functions written solely by him, so some functions in MAIN.R are missing. 

This project uses data from the American Community Survey, the Current Population Survey, and the Financial Health Network Pulse Data. I worked with the ACS and CPS data, so that is what is covered in my code.
